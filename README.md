# Fantasy

An Android application with a backend server. It enables users to explore the virtual world, build buildings, buy equipments, fight monsters, learn skills, add other users as friends while the users are walking around in real world.  
This is the server side of the app. The client side code can be accessed [here](https://github.com/yueyang0115/fantasy-game-client-side).
